# Nachos Docs (Mintlify)

This repository hosts the Nachos documentation site using Mintlify.

## Local preview

```bash
npx mintlify@latest dev
```

## Build for GitHub Pages

```bash
npx mintlify@latest build
```

The static site is generated into the `docs/` directory. Configure GitHub Pages to
serve from the `main` branch and `/docs` folder.
