# Install Overview

Nachos has two dependencies: the **CLI** (manages your stack) and **Docker** (runs your stack). Both are required.

## Requirements

| Dependency     | Minimum version | Check command      |
|----------------|----------------|--------------------|
| Node.js        | 22+            | `node --version`   |
| pnpm           | 8+             | `pnpm --version`   |
| Docker Engine  | 24+            | `docker --version`  |
| Docker Compose | v2+            | `docker compose version` |

## Install path

1. [Install Docker](/install/docker) — get the Docker runtime set up
2. [Install the CLI](/install/cli) — install the `nachos` command globally
3. Run `nachos init` and `nachos up` — see [Getting Started](/start/getting-started)
